# AI Prompt Framework for Local Business Website Copy

## Business Chosen
Cafe Street 1, Greater Noida – Local Café

## Objective
To design reusable AI prompts that generate real, publish-ready website content
for local businesses with adaptable tone and strong CTAs.

## Prompt Logic
- Master prompt sets business context
- Separate prompts for homepage, services, and CTAs
- Tone adaptation supported for different industries

## Tool Used
ChatGPT (Generative AI)

## Output
Complete website copy including homepage, services, and CTA sections
ready for real-world use.
